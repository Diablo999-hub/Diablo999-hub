## ▚ FEATURED_BUILD.log

> ### 🔒 BDS SecureVault — Self-Hosted Password Manager on AWS
>
> Zero-trust Vaultwarden deployment for 50–100 users. Designed, built, and
> shipped **solo, end-to-end**, in ~4.5 hours against a 12-week original scope.
>
> - 🚫 **Zero public inbound ports** — Cloudflare Tunnel only, all traffic outbound
> - 🗄️ **RDS PostgreSQL** with `sslmode=verify-full` against the AWS CA bundle
> - 💾 **Encrypted nightly backups** to S3 with a 30-day lifecycle rule
> - 🔑 **AWS Secrets Manager** — no credential, not even a hash, ever touches git
> - 🛡️ **Least-privilege IAM** — the EC2 role can read exactly one secret
>
> **[→ github.com/Diablo999-hub/bds-securevault](https://github.com/Diablo999-hub/bds-securevault)**
